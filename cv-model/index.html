Done Projects

Portfolio

PatientSentry – Detection System for Abnormal Patient Movements

Development of a real-time hospital patient monitoring system using cameras and the YOLOv8n pose detection algorithm. The project alerts medical staff in case of abnormal movements, helping prevent falls and accidents. Integrated into a Django platform, it enables real-time data collection, transmission, and analysis. It enhances patient safety and proactive intervention. A 3-year business plan shows profitability with a 36.4% average net margin.

Web & Mobile App for Managing Cameroonian Council Certificates

Creation of a web and mobile solution for online management of council certificates (birth, marriage, death). The project is divided into teams, each handling a specific certificate type. It includes front-end and back-end development and seamless integration. The app modernizes administrative services and simplifies certificate access. Agile collaboration ensured smooth progress and testing.

Web Scraping App Deployed on Streamlit

Built a lightweight web application using Python and Streamlit to scrape and display targeted online data. The app extracts and presents real-time information such as listings, articles, or stats. It features a simple and interactive UI for non-technical users. Designed for monitoring and data automation purposes. The project highlights efficient use of web scraping in data-driven tasks.

Blood Bank Management System (Dart / Flutter)

Developed a complete blood bank management system using Dart and Flutter. The app handles donor registration, blood stock tracking, urgent requests, and shortage alerts. It provides a smooth user experience for medical staff and ensures real-time data updates. It also secures donor/recipient data with proper traceability. The system boosts hospital efficiency and response time.
